# Transfer Order System

A web application for uploading images/PDF and processing transfer orders.

## Features
- Upload image or PDF files
- Extract information from QR codes or text
- Generate Excel reports
- Send emails with attachments

## How to Use
1. Visit the application URL
2. Login with username (e.g., 123abc, 456def, 789xyz)
3. Upload your transfer order file
4. Select tray type and quantity
5. Click "Process Transfer Order"

## Technologies Used
- Python Flask
- Pandas
- OpenPyXL
- PyZbar
- Pillow
- PDFPlumber
- Gunicorn
